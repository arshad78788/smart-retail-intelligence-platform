# Smart Retail Intelligence Platform

End-to-end ML project: Customer Segmentation, Churn Prediction, Sales Forecasting, Image Classification.

## Dataset
Download from Kaggle: https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci
Place CSV in data/ folder.

## Modules
- Module 1: Customer Segmentation (RFM + KMeans)
- Module 2: Churn Prediction (LogisticRegression, RandomForest, XGBoost)
- Module 3: Sales Forecasting (ARIMA/SARIMA vs LSTM/GRU)
- Module 4: Product Image Classification (CNN)

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, XGBoost, TensorFlow, Statsmodels, Streamlit

## Setup
1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Download dataset from Kaggle and place in data/ folder
4. Run notebooks in order

## Results
- Customer Segments: 4 (Champions, At Risk, Lost Customers, New/Promising)
- Total Customers Analyzed: 5,878
- Total Revenue Analyzed: $1.77M
- Date Range: Dec 2009 - Dec 2011
